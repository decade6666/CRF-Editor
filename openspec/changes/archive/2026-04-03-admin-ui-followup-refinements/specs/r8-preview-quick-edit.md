# Spec: R8 — 右侧预览双击字段快速编辑

## Scope
前端：预览区双击入口与快捷编辑弹窗。
后端：继续复用现有字段实例更新接口。

## Functional Requirements
- 双击右侧预览中的字段实例可打开快捷编辑弹窗。
- 快捷编辑只是现有字段实例编辑的快捷入口，不引入第二套数据真值。
- 快捷编辑允许修改的属性仅限：
  - `label_override`
  - `bg_color`
  - `text_color`
  - `inline_mark`
- 保存仍走现有字段实例更新接口。
- 保存成功后，主列表与右侧预览立即同步。

## Acceptance Criteria
- [ ] 双击字段可打开快捷编辑弹窗
- [ ] 保存后预览与主编辑区数据立即一致
- [ ] 快捷编辑不引入第二套字段真值来源
