# logos 目录说明

该目录用于存放项目级 Logo 相关说明文档。

## 当前内容
- `README.md`：目录用途说明
- `78c07170-0783-443f-81ab-974481c8fad3.jpg`：现有静态图片资源示例

## 说明
- 该目录中的图片文件属于资产样例或手工整理资源，不参与前端构建输出。
- 运行时项目 Logo 上传文件默认存放在 `uploads/logos/`，由后端接口管理，不写入本目录。
- 项目复制、项目硬删除与 Word 导出使用的 Logo 文件逻辑均围绕运行时上传目录处理。

## 维护建议
- 若需要保留品牌素材，请使用可追踪的命名方式并在此补充来源与用途。
- 若新增静态资源，建议同步说明其使用场景，例如演示截图、品牌素材或文档配图。
